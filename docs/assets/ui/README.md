# UI Assets

Tất cả screenshot phục vụ tài liệu nằm trong `docs/assets/ui/`. Từ bản cập nhật 2025-11-19, chúng ta tách riêng nhóm **retail** để bám sát các màn hình boss cung cấp.

- `./login-gateway.png`, `./dashboard-overview.png`, `./pos-workspace.png` vẫn giữ vai trò placeholder chung.
- **Ảnh chi tiết của KiotViet** → đặt trong [`retail/`](./retail/README.md) và giữ nguyên tên file theo bảng quy ước để markdown không cần chỉnh sửa.

> Luôn dùng ảnh PNG độ phân giải ≥ 1440px chiều ngang, tránh crop chữ/hotline. Nếu nhận ảnh mới, chỉ việc ghi đè file cùng tên.
