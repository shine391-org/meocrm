# 2025-11-19 — Orders Frontend Enhancements

## Summary
- Bổ sung đầy đủ trang `/orders` với bộ lọc (trạng thái, phương thức thanh toán, chi nhánh, ngày), bảng dữ liệu chuẩn hóa và thẻ KPI giúp theo dõi workload theo mô tả tại `docs/reference/07_UI_REFERENCE.md` → phần 10.
- Trang `/orders/[id]` được hoàn thiện để hỗ trợ cập nhật trạng thái workflow PENDING → PROCESSING → COMPLETED, xem chi tiết thanh toán, sản phẩm và thao tác hoàn tiền inline.
- Thêm lớp API client riêng (`apps/web/lib/api/orders.ts`) và tiện ích `lib/orders.ts` để tái sử dụng trong các feature khác (POS bridge, dashboard widget…).
